# Intervention Manager

Plateforme SaaS de gestion d'interventions permettant aux entreprises de gérer leurs techniciens, clients, sites et interventions.

## 🚀 Objectif

Application full-stack développée pour simuler un outil métier utilisé par des sociétés de maintenance.

## 🛠 Technologies

### Backend

- Java 21
- Spring Boot 3
- Spring Security
- PostgreSQL
- Flyway
- Docker

### Frontend

- React
- React Router
- Axios

## 📌 Fonctionnalités

- Authentification JWT
- Gestion des utilisateurs
- Gestion des clients
- Gestion des sites
- Gestion des interventions
- Notifications
- Rapports PDF

## 🏗 Architecture

Backend REST API :

Controller
↓
Service
↓
Repository
↓
PostgreSQL
